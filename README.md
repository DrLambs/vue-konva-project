# vue-konva-project

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

## 注意：需要用一下命令打开谷歌以解决跨域问题
```
open -a /Applications/Google\ Chrome.app --args --disable-web-security --user-data-dir
```
